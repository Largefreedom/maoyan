# nCov2019_crawing.py

### 对新型冠状病毒数据进行爬取，数据来源地址：[丁香园](https://3g.dxy.cn/newh5/view/pneumonia):zap:

---

#### 脚本功能：

* 各国、各省、各地级市地区确诊、疑似、治愈、死亡病例采集，并分为4各工作表分别保存到xlsx文件夹中；

----

#### 脚本使用方法：

* 修改file_path，file_path为excel要存放的目录；
* 直接运行脚本即可；
